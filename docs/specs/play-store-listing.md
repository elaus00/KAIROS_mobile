### 2.2 전체 설명 (4000자 이내)

```
떠오른 생각, 바로 던지면 끝.

Flit.은 기록과 정리 사이의 벽을 허무는 AI 메모 앱입니다.
떠오르는 생각을 그냥 적으세요. 일정인지, 할 일인지, 메모인지 고민할 필요 없습니다.
AI가 자동으로 분류하고, 캘린더와 할 일 목록에 연결합니다.

■ 그냥 적으세요 — Just Capture
앱을 열면 바로 입력창. 텍스트를 적거나 이미지를 첨부하세요.
분류는 AI가 알아서 합니다.

■ AI 자동 분류
적은 내용을 AI가 분석하여 일정, 할 일, 아이디어, 노트로 자동 분류합니다.
태그와 제목도 자동 생성됩니다.

■ 캘린더 자동 연동
일정으로 분류된 내용은 기기 캘린더에 자동 추가됩니다.
캘린더 앱을 따로 열어 수동 입력할 필요가 없습니다.

■ 스마트 노트 관리
AI가 분류한 노트는 폴더별로 자동 정리됩니다.
검색, 태그 필터, 히스토리로 언제든 찾아볼 수 있습니다.

■ 위젯으로 더 빠르게
홈 화면 위젯에서 바로 캡처. 앱을 열지 않아도 됩니다.

■ 깔끔한 디자인
모노크롬 기반의 미니멀 인터페이스. 다크 모드를 지원합니다.

■ 프리미엄 기능
AI 프리셋, 시맨틱 검색, 자동 그룹핑, 분석 대시보드 등
더 강력한 정리 기능을 구독으로 이용할 수 있습니다.

—
Flit.은 더 나은 메모 앱이 아닙니다.
기록을 포기하지 않게 만드는 시스템입니다.

적으면, 알아서 정리됩니다.
```

## 4. 개인정보처리방침 URL — ⬜ 호스팅 필요

Play Store는 **외부 URL**로 개인정보처리방침을 제공해야 합니다.

현재 상태:
- 앱 내 `PrivacyPolicyScreen.kt`에 전체 내용이 존재
- 외부 호스팅 URL은 아직 없음

### 호스팅 옵션

| 방법 | 장점 | 단점 |
|------|------|------|
| **GitHub Pages** | 무료, 즉시 배포 | 별도 레포 또는 /docs 설정 필요 |
| **Notion 공개 페이지** | 무료, 수정 즉시 반영 | URL이 길고 브랜드감 부족 |
| **flit.app 도메인** | 브랜드 일관성 최고 | 웹 서버/호스팅 필요 |

**권장**: `flit.app` 도메인이 있으므로 `https://flit.app/privacy` 형태로 호스팅.
없다면 GitHub Pages로 빠르게 배포 후, 나중에 마이그레이션.

## 8. 사용자 조치 필요 항목

- [ ] **Google Play Console 개발자 계정** 등록 ($25 일회성)
