## Phase 1: MVP — 구현 완료, 통합 테스트 미완

**목표**: 제품의 핵심 가설을 검증할 수 있는 최소 단위

| 기능 | 범위 | 상태 |
|------|------|------|
| 캡처 | 텍스트 입력, 로컬 우선 저장, 완료 피드백, 임시 저장 | **완료** |
| AI 자동 분류 | 유형 판별, AI 제목, 태깅, 엔티티 추출, 사후 수정, Inbox 폴백 | **완료** |
| 홈 화면 | 입력창, 최근 캡처 피드, 히스토리, 검색 | **완료** |
| 공유 인텐트 | 텍스트 수신, 미지원 형식 처리 | **완료** |
| 일정/할 일/노트 | 목록 조회, 삭제, 실행 취소 | **완료** |
| 온보딩 | 앱 소개, 첫 캡처 유도, AI 분류 안내 | **완료** |
| 설정 | 다크 모드, 개인정보 처리방침, 앱 버전 | **완료** |
| 통합 테스트 | 컴파일 통과, 5개 시나리오 검증 | **미완** |

**개발 계획**: `Docs/plan/phase1/`
**진행 보고서**: `Docs/plan/phase1/progress_report.md`

## Phase 2a: 코어 루프 완성

**목표**: 핵심 루프를 일상적으로 사용 가능한 수준으로 완성

| 기능 | 범위 |
|------|------|
| 캡처 | 이미지 첨부 |
| 일정 관리 | 주간/월간 뷰, Google Calendar 동기화 |
| 할 일 관리 | AI 마감일 제안, 우선순위, 순서 변경 |
| 위젯 | 입력 위젯 |
| 알림 | 일정 추가/제안 푸시 |
| 관측 | 분류 품질 최소 관측 지표 수집 |

**개발 계획**: `Docs/plan/phase2a/`

## Phase 2b: 사용성 확장 (최소 출시 기준)

**목표**: 사용성을 확장하고 부가 가치를 추가

| 기능 | 범위 |
|------|------|
| 노트 | 상세 보기, 편집, 검색, AI 자동 그룹화 |
| 위젯 | 할 일 위젯 |
| AI 분류 | 수정 이력 기반 학습 |
| 관측 | 확장 관측 지표 수집 |

**개발 계획**: `Docs/plan/phase2b/`

## Phase 3: 고도화 및 지속 가능성

**목표**: 서비스 지속 가능성 확보. 인증/과금 도입, AI 개인화

| 기능 | 범위 |
|------|------|
| 사용자 인증 | Google OAuth 기반 인증 |
| 구독 모델 | 과금 기준 적용 |
| 일정 관리 | 양방향 Google Calendar 동기화 |
| 노트 | 재정리(구독), AI 시맨틱 검색(구독) |
| AI 분류 | 분류 프리셋(구독), 사용자 지시(구독) |

**개발 계획**: `Docs/plan/phase3/`
